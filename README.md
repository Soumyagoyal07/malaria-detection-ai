# Malaria Detection using Deep Learning
This project detects malaria parasites from microscopic blood cell images using a deep learning model.

## Technologies Used
- Python
- TensorFlow
- Flask
- CNN / MobileNetV2
- HTML CSS JavaScript
## Features
- Upload blood cell image
- AI predicts infected or uninfected
- Modern medical UI
- Real-time prediction
## Dataset
Malaria Cell Images Dataset

## How to Run
pip install -r requirements.txt
python app.py
